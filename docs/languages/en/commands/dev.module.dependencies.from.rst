dev:module:dependencies:from
############################


Show list of modules which depend on %s module



Usage:

.. code-block:: sh

   $ n98-magerun.phar dev:module:dependencies:from [-a|--all] moduleName

Arguments
---------

`moduleName`

  Is required:
     Yes

  Is array:
     No

  Description:
     Module to show dependencies

  Default:
            n/a
