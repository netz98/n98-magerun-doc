customer:create:dummy
#####################


Creates a dummy customers.

Supported Locales:

- cs_CZ
- ru_RU
- bg_BG
- en_US
- it_IT
- sr_RS
- sr_Cyrl_RS
- sr_Latn_RS
- pl_PL
- en_GB
- de_DE
- sk_SK
- fr_FR
- es_AR
- de_AT

Usage:

.. code-block:: sh

   $ n98-magerun.phar customer:create:dummy count locale [website]
