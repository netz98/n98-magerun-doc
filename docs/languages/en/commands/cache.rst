Cache Commands
##############

cache:clean
***********


Clean magento cache

Cleans expired cache entries.
If you like to remove all entries use `cache:flush`
Or only one cache type like i.e. full_page cache:

   $ n98-magerun.phar cache:clean full_page


Usage:

.. code-block:: sh

   $ n98-magerun.phar cache:clean [type]

Arguments
=========

`type`

  Is required:
     No

  Is array:
     No

  Description:
     Cache type code like &amp;quot;config&amp;quot;

  Default:
    n/a
