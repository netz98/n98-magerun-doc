composer:validate
#################


Validates a composer.json

The validate command validates a given composer.json


Usage:

.. code-block:: sh

   $ n98-magerun.phar composer:validate [file]

Arguments
---------

`file`

  Is required:
     No

  Is array:
     No

  Description:
     path to composer.json file

  Default:
            ./composer.json
