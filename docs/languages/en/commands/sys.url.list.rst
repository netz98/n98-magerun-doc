sys:url:list
############


Get all urls.



Usage:

.. code-block:: sh

   $ n98-magerun.phar sys:url:list [--add-categories] [--add-products] [--add-cmspages] [--add-all] [stores] [linetemplate]

Arguments
---------

`stores`

  Is required:
     No

  Is array:
     No

  Description:
     Stores (comma-separated list of store ids)

  Default:
            n/a
    
`linetemplate`

  Is required:
     No

  Is array:
     No

  Description:
     Line template

  Default:
            {url}
